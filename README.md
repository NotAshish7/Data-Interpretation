# IPL Match Prediction & Data Analysis

A comprehensive Microsoft Access-based database system designed to store historical IPL data, perform match analysis, and track user predictions with automated scoring.

## Overview

This project provides a structured environment for analyzing Indian Premier League (IPL) statistics. It utilizes relational database principles to link teams, matches, and venues, allowing for detailed performance tracking and predictive modeling.

## Key Features

* **Match Management:** Centralized database for match schedules, team participants, and results.
* **Predictive Analytics:** A dedicated system for recording user predictions and comparing them against actual outcomes.
* **Performance Tracking:** Automated calculation of prediction accuracy and points earned.
* **Data Visualization:** Built-in charts and reports for analyzing team performance and venue-specific trends.
* **Automated Leaderboard:** A ranking system that identifies top performers based on prediction success.

## Database Components

### Tables
* **Matches:** Stores match details including dates, venues, and winning teams.
* **Predictions:** Tracks user-submitted predictions and calculates points.
* **Users:** Manages participant profiles and cumulative scores.

### Queries
* **Match Summary:** Aggregates overall match results.
* **User Rankings:** Sorts participants by total points for leaderboard display.
* **Accuracy Analysis:** Evaluates the success rate of predictions.

## Requirements

* **Software:** Microsoft Access 2016 or newer.
* **Operating System:** Windows (for optimal Microsoft Access compatibility).

## How to Use

1.  **Download the File:** Clone this repository or download the `.accdb` file directly.
2.  **Enable Content:** Open the file in Microsoft Access and click "Enable Content" if prompted by security settings.
3.  **Data Entry:** Use the standard tables or forms to input new match data or predictions.
4.  **Run Reports:** Navigate to the "Queries" or "Reports" tab to view updated leaderboards and performance charts.
